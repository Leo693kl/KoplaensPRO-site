## О проекте
 
**КомплаенсПро** — первая российская SaaS-платформа для этического и антикоррупционного комплаенса в образовательных организациях. Проект стартовал как MVP для одного университета с архитектурой, рассчитанной на масштабирование до 50+ учреждений.
 
### Зачем это нужно
 
Без выделенного канала жалоб университет работает в режиме «слепого пятна»:
- По данным ECI Global Business Ethics Survey (2021), **79% свидетелей нарушений не сообщают о них** — из-за страха, незнания куда идти, или неверия в результат
- Повторяющиеся нарушения одного сотрудника не связываются между собой
- Конфликты интересов в закупках остаются незамеченными годами
- При отсутствии системы потенциальный ущерб для среднего вуза — **50–150 млн руб./год** (коррупция в закупках + репутационные потери + правовые риски)
### Что делает платформа
 
```
Заявитель → Форма/Telegram-бот → Уникальный ID → Административный кабинет → Расследование → Результат
```
 
- Принимает анонимные обращения о нарушениях через веб и Telegram
- Генерирует уникальный ID для отслеживания статуса без деанонимизации
- Предоставляет комплаенс-офицеру единый кабинет с очередью, фильтрами и аналитикой
- Позволяет вести анонимный диалог между офицером и заявителем
- Строит агрегированную аналитику для выявления системных рисков
